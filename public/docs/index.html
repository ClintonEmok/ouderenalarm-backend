Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Devices

List All Devices

This endpoint retrieves a list of all devices.

Display a listing of the devices.

Example request:
curl --request GET \
    --get "http://localhost:8000/api/devices" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost:8000/api/devices"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Store a newly created device in storage.

Example request:
curl --request POST \
    "http://localhost:8000/api/devices" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"user_id\": 1,
    \"alarm_code\": \"1234\",
    \"longitude\": \"12.345\",
    \"latitude\": \"54.321\",
    \"maps_link\": \"https:\\/\\/maps.google.com\\/?q=12.345,54.321\",
    \"phone_number\": \"+1234567890\",
    \"battery_percentage\": 85
}"

const url = new URL(
    "http://localhost:8000/api/devices"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "user_id": 1,
    "alarm_code": "1234",
    "longitude": "12.345",
    "latitude": "54.321",
    "maps_link": "https:\/\/maps.google.com\/?q=12.345,54.321",
    "phone_number": "+1234567890",
    "battery_percentage": 85
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

POST api/devices

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

user_id   integer   

The ID of the user associated with the device. Example: 1

alarm_code   string  optional  

optional The alarm code for the device. Example: 1234

longitude   numeric  optional  

optional The longitude of the device location. Example: 12.345

latitude   numeric  optional  

optional The latitude of the device location. Example: 54.321

maps_link   string  optional  

optional A Google Maps link for the device location. Example: https://maps.google.com/?q=12.345,54.321

phone_number   string   

The phone number associated with the device. Example: +1234567890

battery_percentage   integer  optional  

optional The battery percentage of the device. Example: 85

longitude   number  optional  

Must be between -180 and 180. Example: -179

Must be between -90 and 90. Example: -89
